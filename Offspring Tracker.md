# Offspring Tracker — App Features

## 1. Parent Account

Parents can create an account and manage their child’s Android device from the app.

Features:

- Parent registration
- Parent login
- Secure authentication
- Parent dashboard
- Manage multiple child devices
- View child device status
- Update parental control rules remotely

## 1.1 Parent and Child Login

The app supports two separate login experiences.

Parent login:

- Parent signs in with email and password
- Parent can register a new parent account
- Parent can pair child devices
- Parent manages app rules, website rules, reports, alerts, subscriptions, and support

Child login:

- Child signs in with child/device name and pairing code
- Child access is created from a parent-paired device
- Child can view their own device status
- Child can view app limits, blocked apps, website rules, and recent alerts
- Child cannot manage or change parent rules

## 2. Child Device Pairing

Parents can connect a child’s Android device with their account.

Features:

- Pair child device using code or QR code
- Add multiple child devices
- Assign device to a specific child
- View paired device information
- Check whether the child device is online or offline

## 3. Installed App Monitoring

Parents can see the apps installed on the child’s phone.

Features:

- View installed app list
- See app name and package name
- Identify social media, games, browsers, and other apps
- View blocked/unblocked status
- Manage app rules from the parent dashboard

## 4. App Usage Tracking

The app can track how much time the child spends on each app.

Features:

- Daily app usage tracking
- Weekly app usage tracking
- Most-used apps report
- Total screen/app usage summary
- Usage history by date
- App-wise usage report

## 5. App Time Limit

Parents can set daily time limits for specific apps.

Example:

- YouTube: 2 hours per day
- Facebook: 30 minutes per day
- Games: 1 hour per day

Features:

- Set daily usage limit for each app
- Automatically restrict app after limit is reached
- Reset app limit daily
- Update time limit remotely
- Show remaining allowed time

## 6. App Blocking

Parents can block specific apps on the child’s device.

Features:

- Block selected apps
- Unblock apps anytime
- Block games, browsers, social media, or any specific app
- Apply app blocking rules remotely
- Log blocked app attempts
- Stronger blocking support using protected/device-owner mode

## 7. Website and Domain Blocking

Parents can block specific websites or domains.

Example:

- example.com
- youtube.com
- facebook.com
- adult websites
- gambling websites

Features:

- Block specific domains
- Block subdomains
- Add/remove blocked websites
- Sync blocked website list from parent account
- Log blocked website attempts
- Local VPN-based filtering support

## 8. Usage Reports

Parents can view detailed usage reports.

Features:

- Daily report
- Weekly report
- App-wise usage report
- Most-used apps
- Blocked app attempts
- Blocked website attempts
- Device activity summary
- Online/offline status report

## 9. Notifications

Parents can receive important alerts.

Features:

- Alert when child exceeds app limit
- Alert when blocked app is opened
- Alert when blocked website is attempted
- Alert when child device goes offline
- Alert when new app is installed
- Remote rule update notification

## 10. Remote Control Rules

Parents can update rules from their own phone.

Features:

- Remotely block/unblock apps
- Remotely set app time limits
- Remotely block/unblock websites
- Sync rules with child device
- Offline rule storage on child device
- Auto-sync when internet is available

## 11. Protected Mode

The app can support a stronger protection mode for child devices.

Features:

- Stronger app blocking
- Stronger uninstall protection
- Device-level parental control
- Better restriction enforcement
- Prevent easy bypassing
- Suitable for serious parental control use

## 12. Child Device Protection Status

Parents can see whether protection is active on the child device.

Features:

- Usage access permission status
- VPN/domain filter status
- Background service status
- Device owner/protected mode status
- Last sync time
- Last online time

## 13. Admin Panel

An admin panel can be added for managing the whole platform.

Features:

- Manage parent users
- Manage child devices
- View platform statistics
- Manage subscriptions
- View reports
- Handle support issues
- Monitor system activity

## 14. Subscription System

The app can include paid plans for parents.

Features:

- Free plan
- Premium plan
- Multiple child device support
- Advanced reports
- Website blocking
- App blocking
- Protected mode support
- Subscription status management

## 15. Main Feature Summary

Offspring Tracker will include:

- Parent login/register
- Child login
- Child device pairing
- Parent dashboard
- Child dashboard
- Installed app monitoring
- App usage tracking
- Daily app time limit
- App blocking
- Website/domain blocking
- Usage reports
- Notifications
- Remote rule control
- Protected mode
- Admin panel
- Subscription system
